# Network Monitoring System

Sistema de monitorização de rede desenvolvido em FastAPI, com suporte para:
- Monitorização de hosts (ping)
- Monitorização de serviços
- Configuração SNMP
- Scheduler automático para verificações periódicas

---

## 🚀 Funcionalidades atuais

- ✔ Registo de hosts (IP, tipo, descrição)
- ✔ Verificação automática de disponibilidade (ping)
- ✔ Associação de serviços a cada host
- ✔ Configuração SNMP por host
- ✔ Dashboard com estado dos hosts
- ✔ Página de detalhe por host
- ✔ Scheduler automático (verificação periódica dos hosts)

---

## 🧱 Tecnologias utilizadas

- Python
- FastAPI
- SQLAlchemy
- PostgreSQL
- Jinja2 (templates)
- Bootstrap (frontend)
- APScheduler (scheduler)
